Expense Tracker

This is a simple project I made to track my daily spending without relying on any external apps. You can add expenses, categorize them, check your total, and edit or delete entries whenever you want. It saves your data so everything stays even after you refresh. The whole point was to keep it minimal, fast, and easy to use.

What it does:
Add expenses with amount, category, and date
Edit or delete entries
Shows total spent
Keeps data saved
Works on both mobile and desktop
